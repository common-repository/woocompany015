== Description ==

<b>WooCompany015</b> permette di ricevere automaticamente gli ordini effettuati sul sito e-commerce, gestito con WooCommerce, nel gestionale COMPANY015 <a href='https://www.company015.it'>https://www.company015.it</a>.<br>
In fase di configurazione puoi decidere se il plugin deve generare un ordine o una fattura in COMPANY015.<br>
Ogni volta che un cliente conclude l'ordine sul sito, automaticamente viene creato il documento che hai impostato in fase di configurazione (ordine o fattura), in questo modo non devi aprire il pannello di WooCommerce
per trasferire l'ordine nel tuo gestionale.<br>
Per ogni nuovo ordine ricevuto dal sito, COMPANY015 ti avvertirà con una notifica.

Il plugin consente di :<br/>
- creare automaticamente l'ordine e/o la fattura nel gestionale;<br/>
- creare la scheda cliente automaticamente;<br/>
- dopo l'attivazione tra il plugin e COMPANY015 è possibile gestire ed aggiornare il catalogo prodotti, direttamente dal gestionale


== Installation ==

<h4>Elementi richiesti</h4>
Per usare il plugin devi avere a disposizione :<br/>

   <ul><li>una licenza COMPANY015 che comprende il modulo "integrazioni WEB"</li></ul>

<h4>Configurazione</h4>
La configurazione di WooCompany015 in wordpress è la seguente:<br>

   <ul>
   <li>vai in Plugins -> Aggiungi Nuovo</li>
   <li>cerca il plugin digitando "WooCompany015"</li>
   <li>fai click su "Installa"</li>
   <li>attiva il plugin</li>
   <li>vai al menu WooCompany015 -> Aiuto per completare la configurazione</li>
   </ul>
